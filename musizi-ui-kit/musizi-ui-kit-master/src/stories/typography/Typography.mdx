# Typography System

The Musizi UI Kit includes a responsive typography system that automatically adapts to viewport size. All typography classes follow a mobile-first approach.

**Accessibility:** Font sizes use `rem` units to respect user browser settings. Line heights use unitless values for proportional scaling.

## Installation

Import the typography CSS in your project:

```js
import '@northgreenug/musizi-ui-kit/typography.css'
```

This will:
- Apply body typography (`Open Sans 1rem/1.5`) to the `<body>` element automatically
- Set default text color to `neutral-800`
- Provide all `mui-text--*` utility classes

## Usage Examples

```html
<!-- Hero section -->
<h1 class="mui-text--display-1">Welcome to Musizi University</h1>
<p class="mui-text--headline-2">Building Tomorrow's Leaders</p>

<!-- Article content -->
<h2 class="mui-text--headline-1">About Our Programs</h2>
<p>Body text is applied automatically to the page...</p>

<!-- Footer -->
<small class="mui-text--small-2">© 2024 Musizi University</small>
```

*Font sizes use `rem` for accessibility. Line heights are unitless.*

## Why rem and unitless line-height?

- **`rem` for font sizes:** Respects user browser settings. If a user sets their browser default to 24px, all text scales proportionally.
- **Unitless line-height:** Prevents inheritance bugs and ensures line spacing scales with the current font size.

## Font Families

The typography system uses two font families:

- **Poppins** (Bold) - Display and Headline styles
- **Open Sans** (Regular) - Body and Small styles

Make sure to import the fonts:

```js
import '@northgreenug/musizi-ui-kit/fonts.css'
```

## Tailwind Utilities

If you're using Tailwind CSS with the Musizi preset, you also have access to individual font size utilities:

```html
<!-- Using Tailwind utilities directly -->
<h1 class="font-poppins font-bold text-display-1-sm md:text-display-1">
  Custom responsive heading
</h1>
```

Available font sizes: `text-display-1`, `text-display-2`, `text-display-3`, `text-headline-1`, `text-headline-2`, `text-headline-3`, `text-body`, `text-small-1`, `text-small-2`, `text-small-3` (and their `-sm` mobile variants).
